# Project Name
Expense Tracker
## Overview
Simple expense tracker app build with react js built-in and hooks, Context API
## Functionality
1. Shows your balance
2. Add & delete transaction
3. Display transactions
## Technologies
- React JS v. 10.2.0
- Context API
- CSS
